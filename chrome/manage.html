About Tiny Spreadsheet Extension

Tiny Spreadsheet Extension (My Tiny Sheet), is an extension that opens an Excel like tiny spread sheet inside your browser. It allows you to sync the spreadsheet online, save data on the cloud and collaborate with others. It is quite usefull for ad-hoc spreadsheet note like taking, short calculations that eventually kept saved on your browser for easy access.

It still lacks many features, but the main functionality is already implemented. Read more why I built it.
