About Tiny Spreadsheet Extension

Tiny spreadsheet extension, is an extension that opens an Excel like tiny sheet inside your browser. It allows you to sync the spreadsheet online, save data on the cloud and collaborate with others. It still lacks many features, but the main functionality is already implemented. Read more about why I built it.
